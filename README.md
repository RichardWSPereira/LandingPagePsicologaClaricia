# LandingPage

Uma landing page estática para o serviço de psicologia online da Psicóloga Claricia. O site apresenta informações sobre a profissional, abordagem terapêutica, benefícios da terapia, depoimentos e formas de contato via WhatsApp.

## Estrutura do projeto

- `index.html` – página principal
- `assets/css/` – estilos do site
- `assets/js/` – scripts de interação
- `assets/img/` – imagens usadas na página
- `assets/video/` – vídeo de apresentação

## Funcionalidades

- Navegação por âncoras para seções da página
- Botões de chamada para agendar sessão via WhatsApp
- Sessão de depoimentos e perguntas frequentes
- Layout responsivo básico com imagens e vídeo

## Como usar

1. Abra `index.html` em um navegador moderno.
2. Verifique se as pastas `assets/css`, `assets/js`, `assets/img` e `assets/video` estão presentes.
3. Para alterar o número de contato, atualize os links `https://wa.me/<número>` nos botões e no rodapé.

## Personalização

- Atualize o conteúdo de `index.html` para adicionar ou editar seções.
- Ajuste os estilos em `assets/css/style.css`, `assets/css/menu.css` e outros arquivos CSS.
- Modifique o comportamento do menu e das seções dinâmicas em `assets/js/createClassToggle.js` e `assets/js/createtag.js`.

## Observações

- O vídeo atualmente está referenciado em `index.html`; confirme se o caminho está correto para o ambiente de hospedagem.
- O projeto não depende de ferramentas de build ou servidor backend.
